Калькулятор ЖКХ — Тверь

Полный опрос и расчёт по всем строкам квитанции.

Данные для расчёта

Холодное водоснабжение

Оценка по ХВС и водоотведению (канализация считается от объёма ХВС; ГВС в этой вкладке не учитывается). Сумма из квитанции — только по этим строкам.

Горячая вода

При счётчике: сначала объём ГВС (м³), затем Гкал на подогрев. Без счётчика объёмы не вводятся — расчёт по нормативу на человека (только число жильцов). Водоотведение в этой вкладке не считается.

Отопление

Только строка отопления. Со счётчиком вводите объём тепла (Гкал). Без счётчика — площадь квартиры (м²), расчёт по нормативу Гкал/м².
