Запускать нужно из директории на уровень выше back.

`go run ./back/cmd/main.go`

Реализованные функции:
- мультиплексорный сервер(вроде так называется). Простофайлики передаёт по путям
- новости. Успешно воруются с одного сайтика

По каталогам. 

/back - общая папка с бэком

/cmd - тут находится точка входа в приложение. Трогать не надо. Разве что параметры.

/internal - внутренняя реализация 

/internal/handlers - обработчики для каждого пути

/internal/server - тут находится класс приложения, который собирает обработчики. Возможно стоит переименовать в app

/static - тут статические файлы

/static/\<page> - все файла страницы

/static/\<page>/index.html - хтмл страницы

/static/\<page>/style.css - стили страницы
